This was my homework for an object-oriented programming course in the 2nd year.

Task: implement a GUI application for managing a high school catalogue in Java,
using the Swing widget toolkit.

Full statement:
http://cs.curs.pub.ro/2013/pluginfile.php/25099/mod_resource/content/1/TemaPOO.pdf
(You might not have access to it as a guest.)

Check out the repo wiki for some screenshots.

pentru fiecare semestru (desi in enunt se cerea sa folosim una singura).

prenume romanesti de pe Wikipedia[1][2].

Faptul ca fereastra nu se poate maximiza cat timp niciun utilizator nu e logat e
un feature, pentru ca am vrut ca imaginea sa acopere fundalul fara a fi
redimensionata.
